# Downloads

Prebuilt archives are published on the
[**GitHub Releases**](https://github.com/pamod-madubashana/Tokex/releases) page.

| Platform | Archive |
| --- | --- |
| Linux x86_64 | `tokex-<version>-linux-x86_64.tar.gz` |
| macOS (Apple Silicon) | `tokex-<version>-macos-arm64.tar.gz` |
| macOS (Intel) | `tokex-<version>-macos-x86_64.tar.gz` |
| Windows x86_64 | `tokex-<version>-windows-x86_64.zip` |

Each archive contains both `tokex` and `rtk`. Extract them into the same directory and run
[`tokex setup`](setup) to configure your provider and key.

Building from source instead? See [Installation](installation).
