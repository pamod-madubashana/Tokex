# Downloads

Prebuilt archives are published on the
[**GitHub Releases**](https://github.com/pamod-madubashana/Cotrex/releases) page.

| Platform | Archive |
| --- | --- |
| Linux x86_64 | `cotrex-<version>-linux-x86_64.tar.gz` |
| macOS (Apple Silicon) | `cotrex-<version>-macos-arm64.tar.gz` |
| Windows x86_64 | `cotrex-<version>-windows-x86_64.zip` |

Each archive contains both `cotrex` and `rtk`. Extract them into the same directory and run
[`cotrex setup`](setup) to configure your provider and key.

Building from source instead? See [Installation](installation).
